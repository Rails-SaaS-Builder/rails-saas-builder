# Changelog

All notable changes to Rails SaaS Builder will be documented in this file.

## [0.9.0] - 2026-02-15

### Added

- Initial public release
- **rsb-settings**: Dynamic runtime settings with schema registry
- **rsb-auth**: Identity & authentication engine with pluggable credential types
- **rsb-entitlements**: Plans, entitlements, usage tracking & payment provider registry
- **rsb-admin**: Admin panel with RBAC, resource DSL, and theme system
- **rsb-entitlements-stripe**: Stripe payment provider (separate gem)
